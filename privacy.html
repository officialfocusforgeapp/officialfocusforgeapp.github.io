FocusForge Privacy Policy

Effective date: March 3, 2026

Privacy Terms

Summary

  • Accessibility is used solely to detect foreground app package names.
  • No typed text, messages, or screen contents are collected.
  • Usage Access is optional and used only for insights features.
  • You may revoke permissions at any time via Android settings.

Information We Collect

Account Data

  • Username, internal ID, role classification.

Blocking Configuration

  • Rules, schedules, severity settings.
  • Event logs such as blocked attempts and achievements.

Accessibility Data

  • Foreground application package name only.
  • No keystrokes, no content capture, no screen scraping.

Optional Usage Access

  • Per-app duration statistics from Android Usage APIs.

Data Storage

Primary data is stored locally on-device. Backend features may store limited account-related data in Firebase infrastructure.

Sharing

  • Firebase for authentication and database services.
  • Google/AdMob where monetization features are enabled.
  • Legal disclosure when required by law.

Data Retention

On-device data persists until deletion or uninstall. Cloud data remains while accounts are active, subject to legal obligations.

Security

Reasonable administrative and technical safeguards are implemented. Absolute security cannot be guaranteed.

International Transfers

Cloud providers may process data in various jurisdictions.
