Privacy Policy

This Privacy Policy explains how FocusForge ("we", "our", "us") collects, uses, stores, and shares information when you use the FocusForge Android app.

Summary

  • We use Android Accessibility only to detect foreground app package names so blocking can work for apps you choose.
  • We do not read typed text, messages, media content, or screen contents.
  • Usage Access (app-time stats) is optional and used only for your insights/projection features.
  • You can disable features or uninstall at any time in Android settings.
No keystroke capture No screen scraping On-device by default Optional Usage Access

1. Information We Collect

1.1 Account and profile data

  • Username, internal user ID, account role (Self/Parent/Child).
  • Profile settings such as avatar, profile line, flair, and in-app preferences.

1.2 Blocking and app configuration data

  • Rules you set for app blocking, schedules, severity, and selected app groups.
  • Events generated by app features (for example blocked app attempts, focus session actions, achievements).

1.3 AccessibilityService data

  • Foreground app package name currently opened on your device (for rule matching/blocking).
  • No typed content, no message contents, and no general screen scraping.

1.4 Optional Usage Access data

  • Per-app foreground duration statistics provided by Android Usage Stats APIs.
  • Used for insights such as top used apps and usage projections.

1.5 Optional social/family/accountability data

  • Friends list, accountability partner selection, linked parent/child relationships, and related settings.
  • Notification/event metadata needed for accountability alerts.

1.6 Ads and monetization data (when enabled)

  • If rewarded ads are enabled, ad providers may process device identifiers and ad interaction data under their own policies.
  • FocusForge only requests ad events needed to run and measure the ad-gate flow.
Accessibility clarification: FocusForge uses Accessibility only to detect the package name of the foreground app so the rules you configure can be enforced. It does not read what you type or view.

2. How We Use Information

  • Provide blocking, focus sessions, progression, and accountability features.
  • Generate app insights and projections when optional Usage Access is enabled.
  • Support family-control and partner-accountability workflows.
  • Detect misuse, enforce policy, and protect service security.
  • Operate beta testing, diagnostics, and product improvements.

3. Where Data Is Stored

Primary app data is stored on-device (DataStore/Room) unless remote backend features are enabled.

If backend mode is enabled, relevant account/social/family data may be stored in Firebase services configured for FocusForge.

4. Sharing of Information

We do not sell personal data. We share data only as required to operate app functionality:

  • Google Firebase (authentication, cloud database, optional backend sync).
  • Google/AdMob (if ads are enabled for rewarded ad functionality).
  • Legal/regulatory disclosure where required by law.

5. Data Retention

  • On-device data remains until you delete it, clear app storage, or uninstall the app.
  • Cloud-backed data is retained while your account and related features are active, subject to legal/business retention obligations.

6. Your Choices and Controls

  • You can disable Accessibility or Usage Access in Android settings.
  • You can modify or remove most app configuration data from within the app.
  • You can request account/data deletion by contacting support.

7. Children and Family Controls

FocusForge includes parent/child account modes and family-control features. Parents/guardians are responsible for lawful use of family features and obtaining required consent for child accounts.

8. Security

We use reasonable administrative and technical controls to protect data. No method of transmission or storage is 100% secure, so absolute security cannot be guaranteed.

9. International Transfers

If cloud services are used, data may be processed in regions where those providers operate infrastructure.

10. Changes to This Policy

We may update this policy from time to time. Updates will be reflected by revising the effective date on this page.

11. Contact

For privacy questions or data requests, contact: officialfocusforgeapp@gmail.com
